#Pré-requis

Node doit être installé (v14.17)

# Front

## Installation des dépendances

```sh
npm install
```

## Lancement du serveur

```sh
npm run dev
```

L'application sera disponible sur : http://localhost:3000


# Back-end

## Installation des dépendances

```sh
npm install
```

## Lancement du serveur

```sh
nodemon server
```

L'API sera disponible sur http://localhost:5000
